# DeepLeEO: Deep Learning for Earth Observation

Functionalities to Deep Learning based classification of Remote Sensing Images

### Dependencies
* numpy
* gdal
* ogr
* tensorflow
* nose
* pillow
* scipy
* matplotlib
* rios
* rsgislib
